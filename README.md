# XMall电子商城  

使用Golang完成一个企业级的电商系统，前端，电商服务端，电商后台管理系统，小程序端，安卓端全部打通。  

## 技术栈  

- 前端：Vue  
- 小程序端：  
- 安卓端：  
- 后端：Go
- Web框架：Gin  
- 数据库：MySQL+Gorm  
- 中间件：jwt-go  
- 缓存：Redis
